# Coffee Sales Dashboard

## Overview
This project is an interactive Excel dashboard developed to analyze coffee sales data and generate actionable business insights.

## Features
- Sales trends analysis
- Top customers identification
- Product performance tracking
- Country-wise sales analysis
- Interactive slicers and filters

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
